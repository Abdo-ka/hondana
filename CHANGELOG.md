# Changelog

All notable changes to Hondana are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Open-source project scaffolding: contribution guide, code of conduct, security
  policy, issue/PR templates, and CI.

## [0.1.0] - 2026-07-11

### Added
- Initial development release of Hondana, a Flutter port of
  [Mihon](https://github.com/mihonorg/mihon).
- Library with categories, sort, tri-state filters, grid/list display, and
  multi-select actions.
- Browse: installable online sources (Madara, MangaDex, TeamX, Themesia, Zeist,
  local), global search, and in-app source WebView.
- Manga details with chapter list and per-chapter download/read state.
- Reader: paged and continuous (webtoon) modes with color filters, grayscale,
  inversion, and blend modes.
- Downloads: background download queue and offline reading.
- History and library-update tracking.
- Settings: appearance (incl. pure-black dark mode), reader, library, downloads,
  browse, data & storage, security (biometric app lock), advanced, and about.
