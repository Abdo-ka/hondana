# Getting help

Thanks for using Hondana! Here's where to go depending on what you need.

## ❓ Questions & how-to

Open a **[GitHub Discussion](https://github.com/Abdo-ka/hondana/discussions)** —
that's the best place for usage questions, ideas, and general conversation.

## 🐛 Bugs

Something broken? **[File a bug report](https://github.com/Abdo-ka/hondana/issues/new/choose)**
with steps to reproduce, your device/OS, and the app version.

## 💡 Feature requests

**[Open a feature request](https://github.com/Abdo-ka/hondana/issues/new/choose)**
and describe the problem you're trying to solve.

## 🔒 Security issues

Please do **not** open a public issue for vulnerabilities — see
[SECURITY.md](SECURITY.md) for private reporting.

## 🧑‍💻 Want to contribute?

See [CONTRIBUTING.md](CONTRIBUTING.md) for the development setup, coding
conventions, commit format, and pull-request process.

> Hondana is a Flutter port of [Mihon](https://github.com/mihonorg/mihon); for
> questions about the original Android app, check the upstream project.
